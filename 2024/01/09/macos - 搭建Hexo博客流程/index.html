macos - 搭建Hexo博客流程

参考文章 https://juejin.cn/post/6978377036645531662
淘宝镜像 npm install -g cnpm –registry=https://registry.npm.taobao.org
是否成功 cnpm -v
安装软件 cnpm install -g hexo-cli
是否成功 hexo -v

下载脚本 curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash
使用版本 nvm install 14.17.0
设初始化 cd ~/Desktop/source/bbb/blog && hexo init
启动博客 cd ~/Desktop/source/bbb/blog && hexo s
是否成功 http://localhost:4000/
创建文章 cd ~/Desktop/source/bbb/blog && hexo n “李老师是个铁憨憨”
编辑文章 cd ~/Desktop/source/bbb/blog && vim source/_posts/李老师是个铁憨憨.md
生成信息 cd ~/Desktop/source/bbb/blog && hexo g
启动服务 cd ~/Desktop/source/bbb/blog && hexo s
下载主题 git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly
https://butterfly.js.org/posts/21cfbf15/#%E6%87%89%E7%94%A8%E4%B8%BB%E9%A1%8C
配置文件 cd ~/Desktop/source/bbb/blog && vim config.yml

复制文件 cd ~/Desktop/source/bbb/blog && cp -rfp themes/butterfly/_config.yml _config.butterfly.yml
下渲染器 npm install hexo-renderer-pug hexo-renderer-stylus –save –verbose
下渲染器 cd ~/Desktop/source/bbb/blog && npm install –save hexo-renderer-jade hexo-generator-feed hexo-generator-sitemap hexo-browsersync hexo-generator-archive –verbose
查看效果 cd ~/Desktop/source/bbb/blog && hexo g && hexo s
新建仓库 新建github公有仓库 shuteng1212.github.io

配置公钥 cat ~/.ssh/id_rsa.pub

最新版本 nvm install node
执行命令 cd ~/Desktop/source/bbb/blog && cnpm install –save hexo-deployer-git
修改配置 cd ~/Desktop/source/bbb/blog && vim _config.yml

配置域名 vim ~/Desktop/source/bbb/blog/source/CNAME

仓库配置 git@github.com/shuteng1212/shuteng1212.github.io.git settings - pages - branch - master /(root)

部署云端 cd ~/Desktop/source/bbb/blog && hexo g && hexo d
是否成功 https://shuteng1212.github.io
是否成功 https://www.shuteng.vip
